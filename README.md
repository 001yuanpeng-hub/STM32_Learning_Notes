# 📘 【STM32外设学习】GPIO 输入与按键消抖

## 一、 本次学习目标
- 掌握 GPIO 浮空输入模式的配置
- 理解按键硬件消抖与软件消抖的区别
- 实现长按与短按功能

## 二、 核心逻辑梳理（🧠唤醒记忆用）
*(这里画图或者写流程，代替冗长的代码)*
1. **时钟使能**：开启 GPIOA 的时钟 (`RCC_APB2PeriphClockCmd`)
2. **引脚配置**：设置为 `GPIO_Mode_IPU` (上拉输入)，因为按键另一端接地。
3. **读取电平**：使用 `GPIO_ReadInputDataBit()` 读取引脚状态。
4. **消抖处理**：检测到低电平后，延时20ms再次检测，依然为低电平则确认按下。

## 三、 关键代码片断
*(只贴核心逻辑，不需要全贴)*
while (1)
{
    if (GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_0) == 0)
    {
        Delay_ms(20); // 软件消抖
        if (GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_0) == 0)
        {
            // 按键确实被按下了，执行任务
            LED_Turn(); 
        }
    }
}

## 四、 🚨 踩坑记录（这部分极其重要！）
- **问题1**：按键有时灵时不灵。
  **原因**：没有开启对应的 GPIO 时钟，或者引脚模式配成了输出来读取。
- **问题2**：松开按键后灯还亮着。
  **原因**：缺少松手检测（while循环卡死在按键按下状态）。
